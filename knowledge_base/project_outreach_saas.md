Flagship project: Full-Stack AI Outreach SaaS. A SaaS that takes a prospect CSV and generates per-contact personalized cold-email sequences. Pipeline: scrape → AI extracts business signals → AI writes a 120-word email per contact → human approval → warmed-up send via Resend → reply detection via Gmail REST → sentiment classification → HubSpot sync. Stack: FastAPI, Supabase (Postgres + Auth + RLS + Realtime), Groq Llama 3.3 70B + 3.1 8B, OpenAI GPT-4o fallback, n8n, Resend, Reoon, HubSpot, React + Vite. Live demo: https://aos-silk.vercel.app/ (read-only, auto-logs in). Status: internal beta — full pipeline validated end-to-end.
